# Sample Nim RxJS app

This sample Nim RxJs app aims to demonstrate how to use React 16.x with Hooks using Nim 1.0.

## Build

`nim js app.nim`

## Run

`npm test`

## Javascript libraries used

- RxJs 6.5.4
- Jest 25.1
